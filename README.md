# AI Recommendation System

## 📌 Overview

The **AI Recommendation System** is an intelligent web application that provides personalized recommendations based on user input using Google's Gemini AI model. The application features a simple and interactive interface built with **Gradio**, making it easy for users to receive AI-generated suggestions in real time.

## ✨ Features

* AI-powered recommendations using Google Gemini
* Interactive and user-friendly Gradio interface
* Fast and accurate response generation

* Lightweight and simple to use

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/shivanimaddela17/AI_Recommendation_sys.git
```

2. Navigate to the project directory:

```bash
cd AI_Recommendation_sys
```

3. Install the required dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

## 📖 How to Use

1. Open the application.
2. Enter your query or input.
3. Click the submit button.
4. Receive AI-generated recommendations instantly.
