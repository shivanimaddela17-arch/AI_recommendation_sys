# AI Recommendation System

An AI-powered Recommendation System that generates personalized suggestions based on user interests using Google's Gemini API. The project uses FastAPI as the backend and Gradio as the frontend to provide recommendations for categories such as movies, books, courses, music, and travel.

## Technologies Used
- Python
- FastAPI
- Gradio
- Google Gemini API

## Run the Project

```bash
pip install -r requirements.txt
python -m uvicorn app:app --reload
python recommad.py
```

Developed for educational and internship purposes.
